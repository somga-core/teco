TeCo (Terminal core) is a C++ game engine for making ASCII-styled games

## Features
- Displaying colored symbols on window or terminal
- Handling separate scenes and ticking current scene
- Managing sprite animations
- Handling key presses
- Using cool effects on symbols displayed on window

## How to use
Detailed instructions on how to use TeCo are in the [`documentation`](https://github.com/somga-core/teco/tree/main/documentation) folder:
- You can check [`quick_start.md`](https://github.com/somga-core/teco/blob/main/documentation/quick_start.md) to start making TeCo projects as fast as possible
- Or start with [`navigation.md`](https://github.com/somga-core/teco/blob/main/documentation/navigation.md), where you can see all articles sorted in order
- Also see [`examples`](https://github.com/somga-core/teco/tree/main/examples/) for some little tests made with TeCo
