TeCo (Terminal core) is a C++ game engine for making ASCII-styled games

## Features
TeCo is capable of:
- Displaying colored symbols on window or terminal
- Handling separate scenes and ticking current scene
- Managing sprite animations
- Handling key presses
- Using cool effects on symbols displayed on window

> [!WARNING]
> If you are using ALT Linux, do **not** use this EnTT installation method.

## How to use
Detailed instructions on how to use TeCo are in the `documentation` folder:
- You can check `quick_start.md` to start making TeCo projects as fast as possible
- Or start with `navigation.md`, where you can see all articles sorted in order
- Also see `examples` for some little tests made with TeCo
